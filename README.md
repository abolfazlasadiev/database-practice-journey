# Database Learning Journey

A structured collection of 100 database projects designed to improve SQL skills and database design knowledge through hands-on practice.

## Levels

- 📁 Beginner (1–20)
- 📁 Basic (21–40)
- 📁 Intermediate (41–60)
- 📁 Upper Intermediate (61–80)
- 📁 Advanced (81–100)

## Topics Covered

- Database Design
- SQL Fundamentals
- CRUD Operations
- Relationships & Foreign Keys
- JOINs
- Aggregate Functions
- Views
- Indexes
- Subqueries
- Stored Procedures
- Triggers
- Performance Optimization

## Structure

```
database-learning-journey/
├── beginner/
├── basic/
├── intermediate/
├── upper-intermediate/
└── advanced/
```

Each project will contain its own SQL files, documentation, and solutions as the journey progresses.

## Goal

Complete all 100 projects to build a solid foundation in relational database design and SQL.
