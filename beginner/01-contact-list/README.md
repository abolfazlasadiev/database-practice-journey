# Contact List Database Project

## Overview
This is a simple SQL project for managing a contact list using MySQL.  
It demonstrates basic database design and CRUD operations.

---

## Features
- Create database and table
- Insert sample data
- Basic SQL queries
- Update and delete records
- Filtering and sorting data

---

## Technologies
- MySQL
- Docker (optional)
